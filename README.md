# Platform Engineering Study — Online Boutique

Um repositório de **estudo prático** para aprender Platform Engineering, DevOps e SRE construindo uma pipeline CI/CD completa. Baseado no [Online Boutique](https://github.com/GoogleCloudPlatform/microservices-demo) da Google.

## 🎯 Objetivo

Montar uma pipeline de CI/CD com foco em:
- **Segurança de supply chain** (SBOM, Cosign, SLSA)
- **GitOps** (ArgoCD / Flux)
- **Observabilidade** (OpenTelemetry, Prometheus, Grafana)
